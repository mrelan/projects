To run the application on a local computer, follow the steps below:
1. Download R from this link https://cloud.r-project.org/ 
2. Download RStudio from this link https://rstudio.com/products/rstudio/download/
3. Open the following file in RStudio, shinyapp/testapp/app.R
4. Click "Run App" on the upper righthand side corner of RStudio

To deploy the application to an online platform, follow the instructions from this link 
https://shiny.rstudio.com/articles/shinyapps.html?_ga=2.151165625.1695120861.1594683356-605255292.1594683356
